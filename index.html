Book Recommender (2024)

Backend Developer (JS)

Attended ZotHacks 2024 after being accepted into it and met my team there for the first time. After much discussion we found a shared interest in reading and decided to create a book recommendation website. Other groups were assigned mentors but ours did not appear so we unfortunately had to do the majority of the work and debugging on our own. We made use of a public library API to access a database of books and their corresponding information. Our front end of the website was made in HTML/CSS while the back end was created with Javascript. The user is able to input certain search criteria (topic, author, rating, etc) and a list of books that fulfill those criteria will be returned or a message indicating that no results for those specific criteria could be found

Checkers AI (2024)

Python Developer (Paired Programming Project)

Developed a Checkers AI with a partner that implements minimax and alpha-beta pruning to search up to 5 moves ahead. Each time it is the AI's turn it generates all possible moves and then simulates future game states, also utilizing a transposition table to reduce redudnancy. Once the maximum depth is reached, the AI evalautes the board with a heuristic scoring function and selects the move with the highest value.

Columns (2023)

Columns Remake | Pygame Developer

Recreated the classic 90s puzzle game Columns in Python using Pygame. Designed core gameplay mechanics including smooth animation, collision detection, and match checking. Players control a falling column of three multicolored jewels, rotating the jewels and moving it to align matching colors. When three or more jewels of the same color connect in any direction, they vanish, and the player continues until the board fills.

Weather Program (2023)

NWS & Nominatim API | Python Developer

Developed a Python application that allows users to request detailed weather information for any U.S. location. After specifying a location, users can query metrics such as maximum/minimum temperature, humidity, and precipitation over a chosen time period. The program returns the requested data along with the nearest reporting weather station. It integrates the National Weather Service (NWS) API and the Nominatim geocoding API to accurately retrieve and map weather data.
